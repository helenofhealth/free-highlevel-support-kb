This article will guide you through how to create and use SSO (Single Sign-On) Magic Links to give clients instant, secure access to individual client portal apps—such as Courses, Communities, or Affiliates—without requiring login credentials. You’ll also learn how to embed these links as custom menu items in the client portal, control access visibility, and streamline navigation for a more professional client experience.

What Are SSO Magic Links?


SSO Magic Links are unique, authenticated URLs that allow users to directly access a client portal app without entering login credentials. When a client clicks their personalized link, they’re instantly logged in and taken to the correct destination—whether it’s a course dashboard, community area, or affiliate portal.


These links are app-specific and contact-specific, meaning each user receives secure access to the exact portal experience they’ve been granted.

Key Benefits of SSO Magic Links


SSO Magic Links are designed to reduce friction, save time, and increase client satisfaction. Key advantages include:


  • One-Click Login: Instant access without entering a username or password.


  • Faster Onboarding: Clients can get started with their app experience right away.


  • Fewer Support Tickets: Eliminates login-related issues such as forgotten passwords.


  • Improved Navigation: Offers a smoother, more direct login experience.


  • Multi-App Convenience: Ideal for setups that include multiple apps like Courses + Communities + Affiliates.


  • Higher Engagement: Easier access encourages repeat visits and ongoing interaction.

How to Generate and Use Magic Links


Magic Links can be created from the Client Portal Dashboard and shared with contacts to provide direct access to specific apps.


Access the Sites Dashboard


Navigate to the Sites tab from the left-hand menu to access your Client Portal Dashboard. This area allows you to manage your client-facing apps and create secure login experiences.




Open the Client Portal Section


Within the Sites menu, click the Client Portal dropdown at the top and select Dashboard. This gives you access to client portal management tools like Magic Link generation and app settings.




Click “Generate” for Magic Link


In the Actions panel, click the Generate button next to “Generate Magic Link.” This begins the process of creating a secure, one-click login for a specific contact and app.




Select Magic Link Type


Choose Custom Menu to generate a reusable magic link intended for client portal navigation. This type of link is ideal for embedding directly in your portal’s sidebar.




Select the App


Use the dropdown under Select App to choose which client portal app the magic link should open—Courses, Affiliates, or Communities. This determines the destination when the link is used.




Generate the Magic Link


Once you’ve selected the app, click the Generate button. This will instantly create a secure, one-click login URL that you can embed or share.




Copy the Magic Link


Once the link is generated, click the copy icon to save it to your clipboard. You can now paste this secure SSO link wherever needed—such as in emails or custom menu items—for seamless client access.


How to Use Magic Links as Custom Menu Links


You can embed Magic Links as custom menu items inside the client portal for quicker access.


To add a Magic Link as a menu item:


Switch to Agency View


From the sub-account dashboard, click the dropdown beside the account name and select Switch to Agency View. This allows you to manage global settings like custom menu links for all sub-accounts.




Access Agency Settings


Once you are in the Agency View, scroll to the bottom of the left-hand sidebar and click on Settings. This is where you can configure platform-wide preferences, including adding custom menu links.




Navigate to Custom Menu Links


In the Settings panel, scroll down and click on Custom Menu Links. This section displays all the existing menu links and is where you can manage or create new custom links for agency and sub-accounts.




Create a New Custom Link


Click the + Create New button in the top-right corner of the Custom Menu Links section to begin creating a new menu link.




Enter Link Details


Fill in the required fields:

  • Link Icon: Choose an icon identifier (e.g., vr-cardboard)

  • Link Title: Name the menu item (e.g., “Test Link”)

  • URL: Paste the full magic link generated previously.




Choose Link Behavior


Select how the link will open when clicked. You can display it inside the current window as an embedded iFrame, open it in a new tab, or load it in the same window.




Set Sidebar Visibility


Choose where the custom menu link will appear. You can display it in the Agency’s sidebar, the Sub-Account’s sidebar, or both for easier navigation across different user levels.




Select Sub-Accounts


Assign the custom menu link to one or more sub-accounts. Only the selected accounts will display this link in their navigation sidebar.




Set Role-Based Visibility


Choose who can access this custom menu link based on user roles. You can limit visibility to Admins, Users, or make it available to all users within the assigned sub-accounts.




Enable Permissions (Optional)


You can grant microphone and camera access to the custom menu link if the embedded content requires these capabilities—ideal for video conferencing or recording tools.




Save the Menu Link


Once you’ve configured all options—such as the URL, sidebar visibility, selected accounts, role-based visibility, and any necessary permissions—click Save to finalize the custom menu link. This will add the Magic Link to the designated sidebar(s), making it easily accessible to users based on the rules you defined.


Frequently Asked Questions


Q: Do Magic Links work for all client portal apps?

They work with Courses, Communities, and Affiliates.


Q: Are Magic Links secure?

Yes. Each link is unique to the contact and app, providing only the necessary access.


Q: Can I use Magic Links in automations or emails?

Yes. You can insert the link in automated messages, onboarding emails, or custom buttons.


Q: Do users need to be logged into the portal first?

No. Clicking the Magic Link logs them in and redirects them automatically.


Q: Can I regenerate a Magic Link?

Yes. Simply repeat the generation process to create a new link.
